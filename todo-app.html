User story:

As a user, I want a mobile application that allows me to add tasks I need to complete to a list. Once the task is complete, I would like to be able to mark it as complete or delete it from my list. The ability to edit the task would also be useful to save time.


Front-end:

  • HTML & CSS
  • JavaScript
  • Mobile Compatible

Back-end:

  • Node.js
  • Express.js
  • JavaScript
  • Passport for user authentication
  • Bcrypt for password security.
  • Form validation

Learning & Challenges:

  • In the process of connecting the application to postgreSQL. User login information is currently being stored in a mimicking database.

Future Additions:

  1. Connect the task list to a database, so that they are stored for future use.
  2. Implement websockets to send alert's to the user's phone as a reminder to complete their tasks.
  3. Add a calender view option to allow the user to create lists on different dates.
  4. Allow the user to create and store mulitple task lists.
